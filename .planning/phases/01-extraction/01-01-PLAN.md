GREEN phase: Create `extractor/cleaning.js` exporting three functions:

1. `cleanTranscript(rawCues)` — Takes array of strings. Pipeline:
   - Strip timestamps: `/\d{1,2}:\d{2}(?::\d{2})?/g` replaced with empty string
   - Filter empty strings after stripping
   - Join with space
   - Strip filler words: `/\b(um+|uh+|you know|like|kind of|sort of|basically|literally|actually|right)\b[,]?\s*/gi` — note the `\b` word boundary to protect "likely", "unlike", etc.
   - Strip speaker labels: `/\[?[A-Z][a-z]+:\]?\s*/g`
   - Smart sentence joining: capitalize letter after `.!?` followed by whitespace
   - Collapse multiple spaces: `/\s{2,}/g` -> single space
   - Trim

2. `countWords(text)` — Returns `{ wordCount: number, isLowWordCount: boolean }`.
   - Split on `/\s+/`, filter empty, count length
   - `isLowWordCount = wordCount < 300`

3. `toFilename(lectureTitle)` — Returns sanitized filename string.
   - Replace `/[^a-zA-Z0-9\s-]/g` with empty
   - Trim, replace `/\s+/g` with hyphens
   - Append `.json`
   - If input empty or result is `.json`, return `'Unknown-Lecture.json'`

Use `module.exports = { cleanTranscript, countWords, toFilename }` for Node.js compatibility (no ES modules — this file also loads in browser via script tag).

Also create `extractor/tests/fixtures/sample-transcript.html` containing a realistic mock of Udemy transcript DOM structure (10-15 cue elements with timestamps and text) for reference in Plan 02.

Run tests again: `node --test extractor/tests/extractor.test.js` — all must PASS.
