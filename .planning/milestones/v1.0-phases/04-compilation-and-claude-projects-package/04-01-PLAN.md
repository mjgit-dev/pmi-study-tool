    **compiler/frontmatter.js** — Single exported function:
    ```javascript
    function parseFrontmatter(fileContent) {
      const match = fileContent.match(/^---\n([\s\S]*?)\n---\n/);
      if (!match) return null;
      const fields = {};
      for (const line of match[1].split('\n')) {
        const colonIdx = line.indexOf(':');
        if (colonIdx === -1) continue;
        const key = line.slice(0, colonIdx).trim();
        const raw = line.slice(colonIdx + 1).trim();
        fields[key] = raw.startsWith('"') ? JSON.parse(raw) : raw;
      }
      return fields;
    }
    ```
    Parse by key name, not position. Handle JSON-quoted strings via JSON.parse, unquoted values (like processedAt) as raw strings. Match ONLY the first `---\n...\n---\n` block (anchored to start of file with `^`).

    **compiler/sections.js** — Two exported functions:
    - `extractBody(fileContent)`: Split at the closing `---\n` of frontmatter. Find the second `---` occurrence. Return everything after `---\n`. If no frontmatter, return the full content.
    - `splitNotesAndQuiz(body)`: Find `'\n## Practice Questions'` in body. If found, `notes` = everything before that index (trimmed), `quiz` = everything from that index + 1 onward (trimmed). If not found, `notes` = full body trimmed, `quiz` = empty string. Do NOT use `'\n## Flashcards'` as a split point — Flashcards is part of the quiz section.

    **compiler/grouper.js** — Three exported functions:
    - `groupBySectionName(lectures)`: Takes array of `{ lectureTitle, sectionName, processedAt, notes, quiz, filename }`. Returns a Map keyed by sectionName. Lectures with falsy sectionName go under `'__unknown__'`. Sort lectures within each group by `processedAt` string comparison (ISO 8601 sorts chronologically). Secondary sort: filename as tiebreaker.
    - `orderSections(groups)`: Takes the Map from groupBySectionName. Returns an array of section names sorted by the earliest `processedAt` timestamp found in each group's lectures. This determines the `01-`, `02-` numbering.
    - `toSlug(sectionName)`: Lowercase, replace `[^a-z0-9]+` with `-`, trim leading/trailing hyphens. Example: `'Introduction to Project Management'` becomes `'introduction-to-project-management'`.

    All modules use `'use strict';` at top, CommonJS `module.exports = { ... }` at bottom. No ES module syntax. No external dependencies.

    Test files use `const { describe, it } = require('node:test');` and `const assert = require('node:assert/strict');` — same pattern as processor/tests/markdown.test.js.
